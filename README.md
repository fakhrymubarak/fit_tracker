# fit_tracker

A weight-tracking platform developed for Android Platform usin Flutter

## Features :

* Sign Up
* Sign In
* Update Profile
* Home Page
* Add Weight List
* Show Weight List
* Sign Out

## Coming Soon Features:

* Take Profile Picture
